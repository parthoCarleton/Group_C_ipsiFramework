Inter Process Service Interface- IPSI framework
IPSI framework is a library that allows multiple applications to connect and perform a service request. This framework abstracts the complex low-level systemâ€™s communication protocol by revealing a simplified service interface that can be used by the applications to invoke service requests to other applications.
